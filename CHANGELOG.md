# Changelog

## [2.0.1] - 2020-05-01

### Fixed

- Fixed error on rendering default value for inputy's options 


## [2.0.0] - 2020-04-27

### Added

- Added high and low intensity flag colors

- Changes options parameter on inputy

- Removed 'D' flag (Dim), not widely supported
